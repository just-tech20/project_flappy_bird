# project_flappy_bird
Game made with Python's Pygame Library

## To Run the Program
Run Command : python flappy_bird.py

## How to Play
Just use SpaceBar to Flap the Bird to go UP while take care not to hit the Pipes.

## Requirements 
Just Need Python 3 to Run the Game
